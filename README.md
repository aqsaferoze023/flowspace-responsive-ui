# flowspace-responsive-ui
# 🚀 FlowSpace - Responsive Productivity Dashboard

A modern and responsive frontend interface built using **HTML, CSS, and JavaScript**.

This project was created as part of a **Full Stack Developer assignment** to demonstrate responsive design principles, semantic HTML, accessibility, and modern UI/UX practices.

---

## 📌 Project Overview

FlowSpace is a productivity dashboard designed with a clean 2025 aesthetic. It provides an engaging and user-friendly experience across all devices.

The project focuses on creating a responsive interface without using any frontend frameworks.

---

## ✨ Features

* 📱 Fully Responsive Design
* 🎨 Modern 2025 UI
* 🌙 Dark/Light Mode Toggle
* 📊 Statistics Dashboard
* 🗺️ Execution Roadmap Timeline
* 💬 Testimonials Section
* 📧 Contact Form
* ⬆️ Scroll To Top Button
* 🧩 Semantic HTML5 Structure
* ♿ Accessibility Friendly
* ⚡ Smooth Animations
